# Letterdrop

Letterdrop は、文字列が画面の上からゆっくり降ってくるリラクゼーション向けの小さなWebアプリです。

静かなフルスクリーン背景の上に、選択した文字セットの言葉がゆっくり流れます。カスタム文字列を保存して、自分用の言葉だけを降らせることもできます。

## 主な機能

- フルスクリーン背景
- 複数の文字列が上からゆっくり降るアニメーション
- プリセット文字セットの切り替え
- カスタム文字列の追加・保存
- 保存済みカスタム文字列の一覧表示
- カスタム文字列の個別削除
- 設定パネルと一覧パネルの右側ドロワー表示

## 使い方

`index.html` をブラウザで開くと動作します。

1. 画面右側の「設定」タブを開きます。
2. 文字セットを選択します。
3. 「カスタム」を選ぶと、追加用のテキストエリアが表示されます。
4. 1行に1つずつ文字列を入力し、「保存して反映」を押します。
5. 保存済みの文字列は、画面右側の「一覧」タブから確認・削除できます。

## カスタム文字列の保存

カスタム文字列はブラウザの `localStorage` に保存されます。

保存後はテキストエリアがクリアされ、文字セットは自動的に「カスタム」になります。次回同じブラウザで開いたときも、保存済みのカスタム文字列が復元されます。

## 技術スタック

- HTML
- CSS
- JavaScript
- Web Storage API (`localStorage`)

外部ライブラリやビルドツールは使用していません。Node.js なしでもブラウザだけで動作します。

## ディレクトリ構成

```text
letterdrop/
├── index.html   # アプリの画面構造
├── styles.css   # 背景、文字アニメーション、ドロワーUIのスタイル
├── app.js       # 文字の生成、プリセット切り替え、カスタム文字列保存処理
└── README.md    # この説明ファイル
```

## 開発メモ

JavaScript の構文チェックを行う場合は、Node.js が利用できる環境で次のコマンドを実行します。

```powershell
node --check app.js
```
